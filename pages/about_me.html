My Developer Journey

My programming journey started back in 2019 during 7th grade. I began by learning simple HTML layouts and experimenting with small programs in VB.NET and Java. Over the years, even if I went on and off with coding, I always found my way back to it. Every project I created, no matter how simple or unpolished, improved my understanding and pushed me toward the goal of becoming a full‑stack developer.

What I'm Good At

I primarily enjoy backend development. I love building systems that work behind the scenes: authentication, session handling, SQL databases, logic workflows, admin-side tools, and anything that requires problem‑solving. I work with PHP, MySQL/SQL, Java, and VB.NET the most, but I'm also comfortable with HTML, CSS, and JavaScript for frontend needs. Lastly, I have also delved into game development using Godot Engine and Unity Engine, creating basic 2D platformer games.

Systems I’ve Built

  • ✔ Digital Enrollment Queuing System (.Net, MySQL, HTML, CSS, JavaScript, PHP)
  • ✔ Marked Attendance System for Teachers (.Net, MySQL)
  • ✔ Calendar Event Scheduling System (.Net, MySQL)
  • ✔ Login & Sign‑up systems with backend + sessions(HTML, CSS, PHP, MySQL)
  • ✔ Smart URL downloader(HTML, CSS, JavaScript, Python with PythonFlask)
  • ✔ Basic 2D platformer action game using Godot(GDScript)
  • ✔ Basic 2D flappy game using Unity(C#)
  • ✔ Basic Login and Sign Up Page using Java(JavaSwing, built in NetBeans DB)

These projects aren't "perfect" or "industry‑level" yet, but they solve practical problems and have been used by real people. That alone motivates me to keep building. I have not included some more projects because I think they are not much and only for school practice.

Where I'm Headed

I want to become a full‑stack developer—someone who can handle both the logic behind the scenes and the interfaces users interact with. I’m still learning, and I know I have a lot to improve. But everything I build takes me one step closer. For now, I’m focused on mastering backend development and growing my frontend skills.

Outside of Coding

When I’m not programming, I enjoy playing volleyball and other active sports, and learning new tech. I may be a student, but I treat every project like a chance to grow.
